# Davi's Chess Game 🏆

A complete professional chess game featuring intelligent AI, realistic wooden design, and multiplayer functionality.

## 🎮 Play the Game

**🔗 Play Online:** https://ricardodelfino.github.io/play-chess/

## ✨ Key Features

### 🧠 Intelligent and Strategic AI
- **3 difficulty levels** powered by enhanced minimax algorithms
- **Aggressive and strategic moves** - attacks, defends, and develops pieces
- **Advanced positional evaluation** considering center control, king safety, and pawn structure
- **No repetitive moves** - each game is unique and challenging

### 🎯 Comprehensive Features
- **Official chess rules** (castling, en passant, promotion, checkmate)
- **3 game modes:** vs AI, local multiplayer, online multiplayer
- **Move history** with algebraic notation
- **Undo moves and pause game**
- **Visual indicators** for valid moves

## 🚀 How to Play

### Basic Controls
- **Click a piece** to select it and view valid moves
- **Click the destination** to move the piece
- **Control buttons:** Undo, Pause, New Game, Settings

### Game Modes
1. **🤖 vs AI:** Choose a difficulty and face a strategic AI
2. **👥 Local:** Two players on the same device
3. **🌐 Online:** Play with friends on different devices using room codes

## 🔧 GitHub Pages Setup

### Automatic Deployment
1. Upload the files to your GitHub repository
2. Go to **Settings** > **Pages**
3. Select **Source:** Deploy from a branch
4. Choose **Branch:** main
5. Click **Save**
6. Wait 5 minutes for deployment
7. Access at: `https://yourusername.github.io/play-chess/`
